# 🚀 Community Discussion Forum with Real-Time Chat

A modern full-stack MERN community platform with:

* 🔥 Real-time chat using Socket.IO
* 📊 Premium analytics dashboard
* ❤️ Likes & comments system
* 🧠 Admin analytics panel
* 📈 Recharts visualizations
* 🎨 Modern SaaS UI
* ⚡ Real-time notifications
* 🌙 Premium dark theme

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Vite
* Tailwind CSS
* Framer Motion
* Recharts
* Lucide React
* Axios
* React Router DOM

## Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* Socket.IO
* JWT Authentication

# 📦 Installation

## Clone Repository

```bash
git clone YOUR_GITHUB_LINK
```

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

## Backend Setup

```bash
cd server
npm install
npm start
```

---

# 🔐 Environment Variables

Create `.env` inside server folder:

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_URL
JWT_SECRET=your_secret_key
```
